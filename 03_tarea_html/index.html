Introducción al lenguaje

HTML (HyperText Markup Language o Lenguaje de Marcado de Hipertexto) es el código estándar utilizado para estructurar y desplegar páginas web y sus contenidos, como textos, imágenes, enlaces y videos. No es un lenguaje de programación, sino un lenguaje de etiquetas que le indica al navegador cómo mostrar la información.

etiquetas de formato

hr: esta etiqueta crea una línea horizontal para separar temáticamente secciones
br: produce un salto de línea forzado dentro de un párrafo o texto.
ambas son elementos vacios (autocerrados)
